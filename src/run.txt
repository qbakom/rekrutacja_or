Prerequisites
1. Python installed (https://www.python.org/downloads/)
2. Git installed (https://git-scm.com/downloads)
3. Packages installed (type in PowerShell "pip install requests")

Running through Visual Studio Code:
Ctrl + ~ (opens terminal in vs code)
Type in terminal:
1. cd src
2. python3 main.py

Running without Visual Studio Code:
1. Find the path of the repository
2. Copy the path to the repository
3. Open PowerShell
4. Type cd "path from clipboard"
5. Type python3 main.py
